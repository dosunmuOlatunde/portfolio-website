FitFuel Landing Page

A modern and responsive landing page for FitFuel, a fitness and healthy lifestyle brand.
This project was built using only HTML5 and CSS3 to practice responsive web design, layout structuring, and modern UI styling.

🚀 Features
Responsive navigation bar
Modern hero section
Features section
Testimonials section
Pricing cards
Footer section
Mobile responsive design
Smooth hover effects
Clean and modern UI
